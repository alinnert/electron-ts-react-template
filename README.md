# Project Template: Electron + TypeScript + React

This is an Electron app template that uses the following technologies:

- Electron **10.1.5**
- TypeScript **4.0.5**
- React **17.0.1**
- React Refresh **0.9.0** (for Hot Reload)
- ESLint **7.13.0** (configured for React Hooks and Prettier)
- Prettier
- Sass **1.29.0**
